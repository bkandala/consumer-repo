FAQ

What is EDS?

Adobe Edge Delivery Services is a composable set of services that enables outstanding experiences with maximum impact.

How do I use shared blocks?

Set libsUrl to the shared libs repo URL in your scripts/scripts.js. All blocks from the libs repo load automatically.

Can I override a shared block?

Yes — create blocks/<name>/ in your own repo. Your local version takes precedence via the externalLibs config option.
